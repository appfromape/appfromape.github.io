Hexo 安裝教學

Hexo 安裝教學

安裝需求

在安裝前您必須先檢查下列您的電腦是否已經安裝下列軟體：

  1. Node.js (Node.js 版本需不低於8.10，建議使用 Node.js 10.0 及以上版本)
  2. Git

安裝 Hexo

$ npm install -g hexo-cli

一旦 Hexo 完成後，請執行下列指令，Hexo 會在指定資料夾中建立所有您需要的檔案。

$ hexo init <folder>

$ cd <folder>

$ npm install

一鰎部署

請參考 官方文檔
